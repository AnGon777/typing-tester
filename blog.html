Here Are Some Important Topic On JavaScript Basics.

Local Storage :

Data does not expire on reload or even if the browser is closed. It stays permanently on the browser until it is removed manually .

Session Storage :

Data expires every time a particular session is closed. It is temporary

Global Scope :

There can be only one global scope on a script. Variables declared in a global scope is accessible from anywhere in the script.

Block Scope :

There can be multiple block scope. It is only accessible within the block of the variable or function it is declared .

Event loop is a JavaScript runtime model. It executes codes, collects and processes data and executes Queued sub-tasks.
